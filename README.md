# 🎓 A Platform for Early Class Dropout Prediction of University Students

## 📌 Overview

Student dropout is a major challenge faced by higher education institutions worldwide. This project aims to identify students who are at risk of dropping out at an early stage by leveraging Machine Learning techniques. By analyzing demographic, academic, and behavioral factors, the system predicts dropout likelihood and helps institutions take timely preventive actions.

The project uses multiple machine learning algorithms, including Multi-Layer Perceptron (MLP), AdaBoost, Stacking Classifier, and Voting Classifier, to achieve accurate predictions and improve student retention rates.

---

## 🎯 Objectives

* Predict student dropout risk at an early stage.
* Analyze demographic, academic, and behavioral factors affecting retention.
* Apply advanced machine learning algorithms for accurate classification.
* Assist educational institutions in implementing early intervention strategies.
* Improve student success and reduce dropout rates.

---

## ❗ Problem Statement

Many universities struggle with high student dropout rates, resulting in academic, financial, and resource losses. Traditional methods often identify at-risk students too late for effective intervention.

This project develops a machine learning-based prediction system that identifies students at risk before dropout occurs, enabling institutions to provide timely support and improve retention.
